# 🩺 Chest X-Ray Pneumonia Detection using CNN

## 📌 Project Overview
This project classifies chest X-ray images into **NORMAL** and **PNEUMONIA** using a Convolutional Neural Network (CNN).

---

## 🧠 Model Approach
- Built a CNN model using TensorFlow/Keras  
- Trained on real-world chest X-ray dataset  
- Evaluated using accuracy, loss curves, and confusion matrix  

---

## 📊 Results
- Training accuracy reached ~98–99%  
- Validation accuracy showed fluctuations due to dataset complexity  
- Confusion matrix shows strong pneumonia detection but weaker normal classification  

---

## ⚠️ Challenges
- Dataset imbalance (more pneumonia cases)  
- Noisy medical images  
- Model bias toward pneumonia predictions  

---

## 🔧 Improvements
- Applied class weighting  
- Used early stopping  
- Tuned classification threshold (0.7) to reduce false positives  

---

## 📌 Key Insight
> In medical AI, understanding model behavior is more important than accuracy alone.
